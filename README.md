# mazinator
Command line tool to generate beautiful maze.

## Instructions

#### If Rust is not installed
Run `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`

#### Install crates and build
`cargo install --path .`

#### To rebuild
`cargo install --force .`

#### Run
`mazinator [number of rows] [number of cols]`
